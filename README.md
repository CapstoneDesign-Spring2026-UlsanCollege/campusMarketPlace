# CampusMarketplace
> **A secure, student-centric trading ecosystem for Ulsan College.**


CampusMarketplace is a dedicated C2C (Consumer-to-Consumer) platform designed to streamline the buying and selling of textbooks, electronics, and dorm essentials within the campus community. By restricting the marketplace to a verified college environment, we prioritize safety and convenience for students.

---

## Key Features

* **Verified Listings:** Create, manage, and track your items with real-time status updates.
* **Smart Search:** Filter items by category, price range, or condition.
* **User Profiles:** Manage your active listings and transaction history in one place.
* **Responsive UI:** Optimized for both desktop browsing and quick mobile checks between classes.
