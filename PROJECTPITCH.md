# 1. Project Title

Campus Marketplace

---

# 2. Problem

University students often need to buy or sell second-hand items such as textbooks, calculators, electronics, or dorm furniture. Existing marketplaces are open to the public and may expose students to scams or unsafe interactions. Students also find it difficult to locate affordable used items within their campus community.

This project aims to solve this problem by creating a **campus-focused marketplace where students can safely trade items with other verified students**.

---

# 3. Target Users

The main users of this system will be **university students** who want to buy or sell used items within their campus community.

Secondary users are International students or exchange students who need temporary items.

---

# 4. What the System Will Do

The system will allow students to create accounts, list items for sale, browse available listings, and contact sellers. Users will be able to search for items and view item details before contacting the seller.

This platform will provide a **simple and trusted environment for students to trade items within campus**.

---

# 5. Core Features (First Version)

The first version of the system will include the following features:

- User registration and login  
- Create and post item listings  
- Browse and search available items  
- View item details  
- Basic messaging between buyers and sellers  

---

# 6. Demo Scenario

**Step 1**  
A user opens the marketplace website.

**Step 2**  
The user creates an account or logs in.

**Step 3**  
The user posts a new item listing (example: a used textbook).

**Step 4**  
Another user searches for textbooks and finds the listing.

**Step 5**  
The second user sends a message to the seller.

---

# 7. Minimum Viable Product (MVP)

The Minimum Viable Product will include a basic web application where students can:

- Register accounts  
- Log in to the system  
- Create item listings  
- Browse available listings  
- View item details  

All data will be stored in a database, and the system will allow users to search for items.

This smallest version demonstrates the **core concept of a student marketplace**.
