# User Stories


## Login Page

- As a student, I want to log in using my email and password, so that I can access my account.  
- As a student, I want to see an error message if my login details are incorrect, so that I can try again.  
- As a user, I want to stay logged in, so that I don’t have to log in every time I visit the site.  

---

## Sign Up Page

- As a student, I want to create an account using my university email, so that I can access the marketplace.  
- As a user, I want to verify my email, so that my account is secure and trusted.  
- As a new user, I want the sign-up process to be simple and quick, so that I can start using the platform easily.  

---

## Home Page

- As a user, I want to see recently posted items on the home page, so that I can quickly explore what is available.  
- As a user, I want to navigate easily to different sections (buy, sell, requests, services), so that I can use the platform efficiently.  
- As a user, I want to see categories on the home page, so that I can browse items based on my needs.  

---

## Language Option

- As a user, I want to change the language of the website, so that I can understand and use the platform comfortably.  
- As an international student, I want to use the platform in English or my preferred language, so that I can easily interact with others.  
- As a user, I want my language preference to be saved, so that I don’t need to change it every time I visit the site.  


## Buying & Browsing

- As a student, I want to browse available items, so that I can find things I need at a lower price.  
- As a student, I want to search items by category (e.g., textbooks, electronics), so that I can quickly find relevant products.  
- As a student, I want to view item details, so that I can decide whether to buy the item.  

---

## Selling Items

- As a student, I want to post items for sale, so that I can earn money from things I no longer need.  
- As a student, I want to add images and descriptions to my listings, so that buyers understand what I am selling.  
- As a student, I want to mark items as sold, so that others know the item is no longer available.  

---

## Communication

- As a buyer, I want to contact the seller, so that I can ask questions or arrange a meeting.  
- As a user, I want to chat with other users, so that communication is easy and fast.  

---

##  Trust & Safety

- As a student, I want to sign up using my university email, so that only verified students can use the platform.  
- As a user, I want to see ratings and reviews of other users, so that I can trust who I am dealing with.  

---

## Item Request Board

- As a student, I want to post a request for an item, so that others can offer it if they have it.  
- As a seller, I want to see item requests, so that I can respond and sell my items faster.  

---

## Services (Tutoring & Skills)

- As a student, I want to offer tutoring services, so that I can earn money by helping others.  
- As a student, I want to find tutors for subjects, so that I can improve my studies.  

---

## Campus Experience

- As a student, I want to choose a campus pickup location, so that I can meet safely and conveniently.
